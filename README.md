## Nodejs Stripe Checkout

Nodejs project for accept pays from stripe

## Setup

```bash
yarn init -y
```

```bash
yarn add express stripe dotenv
```

```bash
yarn add -D @types/express typescript ts-node-dev
```

### add in package.json

```json
"scripts": {
  "dev": "nodemon src/index.ts"
},
```

### first tsconfig

```bash
npx tsc --init
```
